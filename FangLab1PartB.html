      Text editor vs. Word Processor
      A text editor is a program that allows you to create and edit plain text files, while a word processor is a program that allows you to create and edit formatted text documents with various styles and layouts
      Client server model
      A model for how web browsers and web servers communicate with each other over the internet
      FTL
      File Transfer Protocol - A standard network protocol used to transfer files from one host to another over a TCP-based network, such as the internet
      HTTP
      Hypertext Transfer Protocol - The protocol used for transmitting data over the World Wide Web
      TCP/IP
      Transmission Control Protocol/Internet Protocol - A set of protocols that govern how data is transmitted over the internet
